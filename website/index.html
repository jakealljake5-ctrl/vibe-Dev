Quick Installation

Get started with Vibe in seconds. Install the package and import the CSS tokens.

1. Install the package

# Using npm npm install @vibe/core # Or using yarn yarn add @vibe/core

2. Import CSS tokens

import "@vibe/core/tokens";

3. Use components

import { Button } from "@vibe/core"; function App() { return <Button>Click me</Button>; }

Ecosystem

A complete suite of tools and packages to enhance your development workflow.

@vibe/core

Core component library with all the essential UI components you need.

@vibe/icons

Comprehensive icons library with 400+ beautiful icons.

@vibe/testkit

Testing utilities for Playwright to test your Vibe components.

@vibe/codemod

Codemods and CLI tools for seamless migrations and updates.

monday-ui-style

Styling foundations and design tokens (included in @vibe/core).

@vibe/mcp

MCP server for intelligent AI-powered development assistance.

vibe-storybook-components

Vibe Storybook Blocks for building beautiful documentation.

storybook-addon-playground

A Component Playground Addon for Storybook.
